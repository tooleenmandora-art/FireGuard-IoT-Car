# FireGuard IoT Car
IoT Fire Fighting Robot Project

##  Project Overview

FireGuard IoT Car is a smart fire-fighting robot designed to detect and extinguish fires automatically using sensors and embedded systems.

The robot navigates toward the fire source and extinguishes it using a water pumping system controlled by an ESP32 microcontroller.

The project demonstrates the integration of IoT technologies, embedded programming, and hardware components.

---

##  System Features

- Automatic fire detection using flame sensors
- Autonomous robot movement
- Fire extinguishing using a water pump
- Servo-controlled water direction
- Real-time sensor monitoring
- Automatic operation mode

##  System Operation

1. Flame sensors detect fire.
2. The robot moves toward the fire source.
3. The robot stops near the fire.
4. The water pump activates.
5. The servo motor directs the water spray.
6. The fire is extinguished.
